# Terralite Scripting API

Generated by `:tools:generateScriptingApiDocs` from the shared scripting API spec.

## Globals

- `api: ScriptRuntimeApi` - Runtime API available to server-side scripts.
- `StartupEvents: StartupEventsApi` - Startup-only event API used to register content before game data freezes.
- `Registry: RegistryApi` - Startup-only registry API used to inspect or mutate content before freeze.

## Types

### `ResourceId`

Namespaced Terralite resource id, such as `terralite:block/stone`.

```ts
type ResourceId = `${string}:${string}`;
```

## Interfaces

### `ScriptRuntimeApi`

#### Methods

- `info(message: string): void` - Record an informational script diagnostic.
- `onTick(handler: (tick: ServerTick) => void): void` - Run a handler once per authoritative server tick.
- `world(): ServerWorldApi` - Access world operations exposed to server scripts.

### `ServerTick`

#### Properties

- `readonly index: number` - Monotonic tick index.
- `readonly deltaMillis: number` - Milliseconds elapsed during this tick.
- `readonly totalMillis: number` - Milliseconds elapsed since the server script host started.

### `ServerWorldApi`

#### Methods

- `entityCount(): number` - Number of entities currently known to the script world view.
- `chunkCount(): number` - Number of loaded chunks currently known to the script world view.
- `hasChunk(x: number, y: number, z: number): boolean` - Returns true when the chunk at the given chunk coordinates is loaded.
- `loadChunk(x: number, y: number, z: number): boolean` - Load or create the chunk at the given chunk coordinates.
- `unloadChunk(x: number, y: number, z: number): boolean` - Unload the chunk at the given chunk coordinates.

### `BlockBuilder`

#### Methods

- `displayName(value: string): this` - Set the player-facing block name.
- `solid(value: boolean): this` - Set whether this block has full solid collision.
- `translucent(value: boolean): this` - Set whether this block should be treated as transparent.
- `material(value: string): this` - Set the block material key, such as `stone`, `dirt`, or `plant`.
- `hardness(value: number): this` - Set break hardness; must be non-negative when content is built.
- `resistance(value: number): this` - Set explosion resistance; must be non-negative when content is built.
- `requiresTool(value: boolean): this` - Set whether the block requires an appropriate tool.
- `soundType(value: string): this` - Set the sound type key.
- `category(id: ResourceId): this` - Add this block to a creative category.
- `tag(id: ResourceId): this` - Add this block as a member of a tag.
- `model(path: string): this` - Placeholder for future block model path support. Currently chainable only.
- `texture(path: string): this` - Placeholder for future block texture path support. Currently chainable only.
- `renderType(type: string): this` - Placeholder for future block render type support. Currently chainable only.
- `color(r: number, g: number, b: number): this` - Placeholder for future block color support. Currently chainable only.
- `opacity(value: number): this` - Placeholder for future block opacity support. Currently chainable only.
- `tintKey(value: boolean): this` - Placeholder for future tint support. Currently chainable only.
